# The F2E 總統副總統大選開票地圖

## 作品說明
- 用於查詢總統副總統大選資料
- 可透過點選地圖或選項來檢視市、區、里資料分布
- 可在PC、行動裝置上友善的獲得選舉資訊

## 系統說明
專案運行方式是使用 yarn install、yarn run dev

## 資料夾說明
- views - 畫面放置處
- components - 通用元件及畫面元件放置處
- assets - 樣式、icon及圖片放置處
- stores - 資料狀態放置處
- utils - 函式資料放置處
- public - 選舉csv資料放置處

## 使用技術
- vite
- vue
- tailwind
- SCSS
- CSS Flexbox
- SVG
- RWD
- Chart.js
